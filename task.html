Task

For comparing the tools we set up the application myTinyTodo, which allows users to manage todo lists.


The application needs the following requirements:


- Apache Web Server

- PHP 5.2.0 or greater;

- PHP extensions: php_mysql (MySQL version)


We set up the application with all tools on two servers (one with the application, one with the database).

As operating system Ubuntu 14.04 was used and the servers were set up on an openstack system.


Further information about the myTinyTodo application can be found here:


http://www.mytinytodo.net/
